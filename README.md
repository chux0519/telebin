telebin is a simple bot for saving your DM into db(sqlite).


## usage

1. start the server

> export TELEGRAM_BOT_TOKEN='XXXXXXXXXX:xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx_XXX'
> 
> export DATABASE_URL="sqlite:telebin.db"
> 
> cargo run

2. send message to bot
